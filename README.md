## 简介

Shadowrocket 是一款功能强大的网络代理工具，支持多种代理协议，为用户提供安全、稳定的网络访问体验。本版本为 macOS 平台优化版本。

## 主要功能

### 核心功能
- **多协议支持**：支持主流代理协议，满足不同场景需求
- **智能路由**：根据规则自动选择代理或直连
- **速度测试**：内置延迟测试和速度测试功能
- **流量统计**：实时显示上传下载流量
- **节点管理**：支持多节点配置和快速切换
- **规则订阅**：支持在线规则订阅和自动更新

### 高级功能
- **分流规则**：支持基于域名、IP、地理位置的智能分流
- **广告拦截**：内置广告过滤规则
- **DNS 设置**：支持自定义 DNS 服务器
- **网络诊断**：提供网络连接诊断工具
- **日志记录**：详细的连接日志便于问题排查
- **场景模式**：支持多种使用场景配置

## 使用教程

### 快速开始

#### 1. 添加服务器节点

**方法一：手动添加**
- 点击右上角 "+" 按钮
- 选择协议类型（如 Shadowsocks、VMess 等）
- 填写服务器信息：
  - 服务器地址
  - 端口
  - 密码/UUID
  - 加密方式
  - 其他协议相关参数

**方法二：扫描二维码**
- 点击左上角扫描图标
- 扫描服务器提供的二维码
- 自动导入节点配置

**方法三：导入订阅链接**
- 点击 "订阅" 菜单
- 点击 "添加订阅"
- 粘贴订阅 URL
- 点击 "完成" 并更新订阅

#### 2. 选择节点

- 在节点列表中选择要使用的服务器
- 点击节点旁边的延迟测试图标测试速度
- 选择延迟最低的节点以获得最佳性能

#### 3. 启动代理

- 点击顶部的连接开关
- 首次使用需要授权网络扩展权限
- 输入 macOS 用户密码
- 连接成功后图标变为彩色

#### 4. 配置规则

**全局模式**
- 所有流量通过代理服务器

**规则模式**（推荐）
- 根据规则智能分流
- 国内网站直连，国外网站走代理

**直连模式**
- 所有流量不经过代理

### 高级配置

#### 自定义规则

1. 点击 "配置" 标签
2. 选择 "编辑配置"
3. 添加自定义规则：
   ```
   # 直连规则示例
   DOMAIN-SUFFIX,baidu.com,DIRECT
   DOMAIN-SUFFIX,qq.com,DIRECT
   
   # 代理规则示例
   DOMAIN-SUFFIX,google.com,PROXY
   DOMAIN-SUFFIX,youtube.com,PROXY
   
   # IP规则示例
   IP-CIDR,192.168.0.0/16,DIRECT
   GEOIP,CN,DIRECT
   
   # 最终规则
   FINAL,PROXY
   ```

#### 订阅规则

1. 添加规则订阅 URL
2. 定期更新以获取最新规则
3. 支持多个规则源组合使用

#### DNS 配置

- 设置 > DNS 设置
- 推荐使用：
  - `223.5.5.5` (阿里DNS)
  - `119.29.29.29` (腾讯DNS)
  - `8.8.8.8` (Google DNS)
  - `1.1.1.1` (Cloudflare DNS)

## 常见问题

### 无法连接？

1. 检查服务器节点信息是否正确
2. 测试节点延迟，选择可用节点
3. 尝试切换不同的协议或端口
4. 检查本地网络连接是否正常
5. 查看日志排查具体错误

### 速度慢？

1. 进行速度测试，选择最快的节点
2. 切换到离你更近的服务器
3. 尝试不同的协议（如 VMess、Trojan）
4. 检查本地网络带宽是否受限

### 部分网站无法访问？

1. 检查是否处于规则模式
2. 添加对应网站到代理规则
3. 尝试切换到全局模式测试
4. 清除浏览器缓存和 DNS 缓存
